# Extreme weather module
The extreme weather use a baseline cost, presented as a percentage of GDP, for each scenario. 
This baseline cost is then linearly scaled based on the adjusted temperature increase projected for each scenario.

Baseline costs are found under `data/pattern_scaling/extreme_weather_costs.csv`.
